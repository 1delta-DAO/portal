# Allocator UI by 1delta

Simple UI that uses 1delta APIs to display lending data and allow basic lending operations.

## UI

The ui is in root, you need to set the related environment vars for ui first, the example `.env` is [.env.example](ui/env.example)
Then install the required packages and start the web app

### Use direct EOA Caller

```bash
pnpm i && pnpm start
```
